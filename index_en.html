This is a preliminary list. I'll add constantly more data, enrich it and remix it. The goal of this page is to present the corrected xmls, as some of the data is not exactly W3C and sometimes - downright broken as a format and usability. I'll add some JSON and CSV once the schema stabilises. We are still expecting the big hit - the indexed and linked plenary meeting records.

This data is public domain. You can use it, remix it, enrich it and pass it on.
